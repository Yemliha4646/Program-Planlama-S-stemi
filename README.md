# Sınav Geri Sayım ve Dinamik Çalışma Kartları

Python / FastAPI tabanlı, Clean Architecture prensiplerine uygun bir sınav takip ve flashcard uygulamasıdır. SQLite ve SQLAlchemy ile veritabanı yönetimi sağlanır. Frontend, Tailwind CSS ile yazılmış tek sayfalık bir Vanilla JavaScript arayüzü içerir.

## Başlatma

1. Sanal ortam oluşturun ve etkinleştirin.
2. `pip install -r requirements.txt`
3. `python main.py`
4. Tarayıcıda `http://127.0.0.1:8000` adresini açın.

## Özellikler

- Sınav ekleme, silme
- Flashcard ekleme ve yönetme
- Günlük hedef hesaplama
- Leitner sistemi ile "Bildim" / "Bilemedim" güncelleme
- Tailwind CSS ile modern ve responsive arayüz
